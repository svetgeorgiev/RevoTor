Teensy 4.1 Flight Computer -> main.ino and main_simple.ino - Weather Balloon Flight Computer - 

The current example contains: 

- Teensy 4.1 - MCU 
- LoRa RFM95W - Telemetry
- uBlox M10Q - GPS 
- BMP085 - Barometer
- MPU6050 - Accelerometer / Gyroscope
- HMC5883L - Magnetometer / Compass
